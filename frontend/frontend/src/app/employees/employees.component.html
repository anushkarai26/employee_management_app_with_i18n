How does the Employee Management Application Work?

Angular 18's AuthGuard restricts access to certain routes or components based on the roles assigned to the authenticated users.

  • If the user's role is Admin, they will be able to add and update the Employee database.
  • If the user's role is User, they'll be able to view Homepage and their Profile but will be denied access to manage the Employee database.
  • If the user is not logged in, they will be denied access and would have to login first.
